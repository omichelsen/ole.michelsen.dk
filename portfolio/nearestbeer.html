2012 · Founder · Design · Programming: HTML5, JQuery Mobile, MVC4

I founded Nearest Beer, a social app which helps you find your favourite brew served nearest you.

The app has integration to Fourquare, Google Maps and Facebook OAuth.

Backend is implemented in C# using ASP.NET MVC4 and Entity Framework 5. I'm using a code first approach with a generic repository pattern, a service layer and Ninject for dependency inejction.
