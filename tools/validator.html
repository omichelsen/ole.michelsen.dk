Tools

This tool validates a range of URI's against the official W3C Markup Validation Web Service. Simply enter the URI of your sitemap file, and this tool will automatically read and validate the linked pages from the file. The sitemap must be in the Sitemap or RSS feed format.

In accordance with best practices for consuming remote resources, this tool will only process one URI per second so not to overload the validator web service. Furthermore it will not process more than 30 URI's. I will perhaps revise this cap later, when I see how it performs.
